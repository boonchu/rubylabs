== README

* Scaffolding Generator
* Applying Scaffolding to DB
* start rails and update DB
* run test to retrieve records from MVC

```
rails generate scaffold car make color year:integer
rake --task
rake db:migrate
rails s
curl http://localhost:3000/cars.json
```
